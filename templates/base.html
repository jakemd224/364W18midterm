Description

This application is designed to allow users to create and edit 5 different parties (teams) of Pokemon.
Pokemon may not be reused so that each party is entirely unique.

Navigation

  • Go to Pokemon input form.
  • Go to battle input form.
  • See all Pokemon that have ever been added.
  • See all users that have used this web app.
  • See all parties that have been created.
  • See all battles that have taken place.
